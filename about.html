About This App

Purpose

This To-Do App helps users manage their daily tasks efficiently with a clean and intuitive interface.

Instructions

  1. Use the input field on the Home page to add new tasks
  2. Click on tasks to mark them as complete
  3. Use the delete button to remove tasks you no longer need
  4. Change task status using the dropdown selector
